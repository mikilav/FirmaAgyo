Descrizione dettagliata

Gestisce la richiesta di controllo dello stato di 1 o più documenti caricati in Agyo (hubid presente) Le opzioni per l'invio e la firma sono definite nei file guida elencati nel file dacbl di input.

Il risultato dell'operazione viene scritto nei file di output acbl.
