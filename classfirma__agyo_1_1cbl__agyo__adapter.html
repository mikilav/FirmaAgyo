Descrizione dettagliata

Gestisce l'invio di uno o più documenti ad Agyo per le operazioni relative alla firma elettronica.

Le opzioni per l'invio e la firma sono definite nei file guida elencati nel file dacbl di input. Il risultato dell'operazione viene scritto nei file di output acbl.
